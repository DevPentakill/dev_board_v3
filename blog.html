Question-1: What are the different ways to select an element in the DOM?

We can use few methods to select DOM elements like getElement by ID, TAG, Classname and also we have query selectors

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Mainly these are used to get access to the inside contents of an element. Inner html gives the whole html element whether its commented or not, inner text usually just gives the text that is viewed in the website and text content is used to get everything from script tags to style tags i.e. anything.

Question-3: What is event delegation in the DOM?

Event delegation in dom refers to the distribution actions that each DOM nodes or elements will resopond to by the user

Question-4: What is event bubbling in the DOM?

When we trigger an event in JS dom, there occurs a phenemenon called event bubbling. THis basically goes from target elements to parent elements

This question's answer needs a broad explanation, but in short we have different methods that we can use in our DOCUMENT object like create element, appendchild and remove to manipulate the data we want to show in our website.
